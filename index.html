微软MicroSoft | 云端创客

发表于 2016-07-30

Project Name：云端创客 微软携手机智云 IoT 开发者夏令营

Timer：2015年6月10日 下午 01:00 ～ 2015年6月10日 下午 06:00

Add：上海市徐汇区虹桥路3号港汇大厦2座 微软（中国）上海分公司 10层大会议室

Describe：IOT 和智能硬件大潮风起云涌，微软发布的 Windows 10 IOT Core 以及 Azure 云平台针对 IOT 场景的多种服务模块（Event Hub、Stream Analytics、Power BI）可帮助各类创客打开无限想象空间。

本系列云端 IOT 开发夏令营将通过主题演讲、演示和动手实验相结合的方式为开发者和创客们带来微软 IOT 领域的全新体验。以植物宝为例，演示如何使用树莓派和 GoKit 控制外设及完成对植物生长环境的监控，并实现对相关监控数据到云端的上传、存储及分析。再辅以机器人小车的动手实验，实现小车上传感器的感知、IOT 系统的连接、数据运算处理，从而最终实现对小车运动方向的实时操控。
